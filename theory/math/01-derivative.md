# Производная — Теория от нуля до понимания

---

## Почему производная существует — проблема, которую она решает

Допустим, машина едет из Москвы в Петербург. За 4 часа она проехала 400 км.
Средняя скорость = 400 / 4 = **100 км/ч**.

Но это средняя. В какой-то момент машина стояла в пробке (0 км/ч), потом разогналась до 140 км/ч на трассе. Спидометр показывает **мгновенную скорость** — скорость прямо сейчас, в данную секунду.

Как вычислить «прямо сейчас»? Нельзя взять интервал 0 часов — деление на ноль. Но можно взять очень маленький интервал Δt → 0.

**Производная — это мгновенная скорость изменения функции в точке.**

---

## Геометрический смысл

Возьми функцию y = x². Нарисуй её мысленно — парабола.

Если взять две точки на параболе и провести через них прямую — получишь **секущую**. Наклон этой прямой показывает, как быстро функция менялась **в среднем** между двумя точками.

Теперь представь, что ты двигаешь вторую точку всё ближе к первой. Секущая начинает вращаться. В пределе, когда вторая точка совпала с первой, секущая превратилась в **касательную** — прямую, которая касается кривой ровно в одной точке.

**Производная в точке x = a — это угловой коэффициент (наклон) касательной к графику в точке (a, f(a)).**

Крутой наклон = функция быстро растёт.
Нулевой наклон = функция в этой точке на вершине или в яме (локальный максимум/минимум).
Отрицательный наклон = функция убывает.

---

## Формальное определение через предел

Наклон секущей через точки (x, f(x)) и (x+h, f(x+h)):

```
Δy / Δx = [f(x + h) − f(x)] / h
```

Производная — предел этого выражения при h → 0:

```
f'(x) = lim[h→0] [f(x + h) − f(x)] / h
```

Это записывают несколькими способами — все они обозначают одно и то же:

```
f'(x)       ← обозначение Лагранжа, самое компактное
dy/dx       ← обозначение Лейбница, подчёркивает «скорость изменения y по x»
Df(x)       ← оператор дифференцирования
ẋ           ← обозначение Ньютона, используется для производной по времени
```

---

## Вычисляем производную x² из определения (один раз — чтобы понять)

```
f(x) = x²
f(x + h) = (x + h)² = x² + 2xh + h²

[f(x + h) − f(x)] / h = [x² + 2xh + h² − x²] / h
                       = [2xh + h²] / h
                       = 2x + h

При h → 0:  2x + h → 2x
```

Итого: **(x²)' = 2x**

Смысл: в точке x = 3 наклон касательной к параболе y = x² равен 2·3 = 6. В точке x = 0 — наклон равен 0 (вершина параболы, горизонтальная касательная).

## Правила дифференцирования

### Правило константы
```
(c · f(x))' = c · f'(x)
```
Пример: (5x³)' = 5 · 3x² = 15x²

### Сумма и разность
```
(f ± g)' = f' ± g'
```
Пример: (x³ + 2x − 7)' = 3x² + 2

### Произведение (правило Лейбница)
```
(f · g)' = f'g + fg'
```
Пример: (x² · sin x)' = 2x · sin x + x² · cos x

Запомни как: «производная первого умножить на второго, плюс первый умножить на производную второго».

### Частное
```
(f/g)' = (f'g − fg') / g²
```
Пример: (x²/sin x)' = (2x · sin x − x² · cos x) / sin²x

### Цепное правило (сложная функция)
```
(f(g(x)))' = f'(g(x)) · g'(x)
```
Пример: (sin(x²))' = cos(x²) · 2x

Думай о нём так: дифференцируешь внешнюю функцию (не трогая то, что внутри), затем умножаешь на производную того, что внутри.

Ещё пример: (e^(3x))' = e^(3x) · 3 = 3e^(3x)

## Физический смысл — три уровня

Если s(t) — положение тела в момент времени t:

```
s(t)    — положение (где тело)
s'(t)   — скорость (как быстро меняется положение)
s''(t)  — ускорение (как быстро меняется скорость)
```

Пример:
```
s(t) = t³ − 6t² + 9t + 2

s'(t) = 3t² − 12t + 9   ← скорость
s''(t) = 6t − 12         ← ускорение

При t = 2: s'(2) = 12 − 24 + 9 = −3  (тело движется назад!)
При t = 2: s''(2) = 12 − 12 = 0     (ускорение равно нулю — скорость не меняется в этот момент)
```

## Производная и экстремумы

Если f'(a) = 0 — касательная горизонтальна. Это **критическая точка**: потенциальный максимум, минимум или точка перегиба.

Как понять что именно:
- Если f'' (a) > 0 → минимум (функция «улыбается» ∪)
- Если f'' (a) < 0 → максимум (функция «хмурится» ∩)
- Если f'' (a) = 0 → нужны дополнительные проверки

Пример: y = x² − 4x + 3
```
y' = 2x − 4 = 0  →  x = 2  ← критическая точка
y'' = 2 > 0      →  минимум

y(2) = 4 − 8 + 3 = −1
Минимум функции: точка (2, −1)
```

## Частые ошибки

1. **Забывают цепное правило**: (sin(2x))' ≠ cos(2x). Правильно: cos(2x) · 2
2. **Путают (f·g)' с f'·g'**: это не одно и то же. Правило произведения: f'g + fg'
3. **Берут производную константы как 1**: (7)' = 0, не 1.
4. **Знак при производной косинуса**: (cos x)' = **−**sin x. Минус часто теряют.

## Как объяснить производную своими словами (для заучивания)

> Производная функции f в точке x — это число, показывающее, насколько быстро функция меняется в этой точке. Геометрически это наклон касательной к графику. Физически — мгновенная скорость. Формально — предел отношения приращения функции к приращению аргумента, когда приращение аргумента стремится к нулю.

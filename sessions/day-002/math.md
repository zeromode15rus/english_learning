## Математика — Производная (The Derivative)

### Интуиция

Скорость автомобиля в конкретный момент времени — не средняя скорость за поездку, а скорость *прямо сейчас*. Производная — это математический инструмент, который делает это понятие точным: мгновенная скорость изменения функции в точке.

Способ построить её: возьми два близких значения x и x + h, посмотри, как изменилась функция, раздели изменение на размер шага — и устреми шаг к нулю.

### Формальное определение

$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$

Геометрический смысл: f'(x) — это угловой коэффициент (наклон) касательной к графику f в точке x. Если касательная идёт вверх — f'(x) > 0; горизонтальна — f'(x) = 0; вниз — f'(x) < 0.

Физический смысл: если f(t) — положение тела в момент t, то f'(t) — мгновенная скорость, f''(t) — ускорение.

### Вычисление: пример из определения

Найдём f'(x) для f(x) = x²:

$$\frac{f(x+h) - f(x)}{h} = \frac{(x+h)^2 - x^2}{h} = \frac{x^2 + 2xh + h^2 - x^2}{h} = \frac{2xh + h^2}{h} = 2x + h$$

При h → 0: f'(x) = **2x**.

### Основные правила (без вывода)

| Правило | Формула |
|---|---|
| Степенное | (xⁿ)' = nxⁿ⁻¹ |
| Суммы | (f + g)' = f' + g' |
| Произведения | (fg)' = f'g + fg' |
| Частного | (f/g)' = (f'g − fg') / g² |
| Цепное | (f(g(x)))' = f'(g(x)) · g'(x) |

### Три конкретных примера

**1.** f(x) = 3x³ − 5x + 2
Применяем степенное правило и правило суммы:
f'(x) = 9x² − 5

При x = 2: f'(2) = 9·4 − 5 = **31** — наклон касательной в точке x = 2.

**2.** g(x) = sin(x²) — цепное правило:
Внешняя функция: sin(u), её производная cos(u). Внутренняя: u = x², производная 2x.
g'(x) = cos(x²) · 2x

**3.** Нахождение минимума: f(x) = x² − 4x + 7
f'(x) = 2x − 4 = 0 → x = 2. Это точка минимума (f''(x) = 2 > 0 — функция выпукла).
f(2) = 4 − 8 + 7 = **3** — минимальное значение.

### Связь с градиентным спуском

В машинном обучении веса обновляются по правилу:
w := w − α · ∂L/∂w

∂L/∂w — это производная функции потерь L по весу w (частная производная = «градиент по одному весу»). Производная говорит, в каком направлении функция растёт; мы движемся в противоположном.

### Распространённые ошибки

- Путать (fg)' = f'g' (неверно!) с правилом произведения (f'g + fg')
- Забывать цепное правило: производная sin(3x) — это не cos(3x), а cos(3x) · 3
- Считать, что f'(x) = 0 всегда означает минимум: это может быть максимум или точка перегиба
